## [Blend-shop](https://zviacheslavv.github.io/blend-shop/)

> A small educational project simulating a simple online shop.

---

## Tech Stack
- **HTML, CSS, JS** core web technologies
- **Vite** fast dev server and build tool
- **Libraries**: Axios (HTTP) requests, izitoast (notifications)

## Key Features
- Display products and basic shop interface
- Adding products to wihlist and cart
- Saving user's choices in local storage
- User notofocations for actions
- Fast development reload with Vite
- BEM methodology
